Release Notes

Significant new features and known issues for each release are listed below. To report a new issue, or to see the complete list of outstanding issues, please use the pvOSPRAY issue tracker.

Alpha 2 Release (2015-07-22)

  • New Features
    • Accurate transparency without depth peeling
    • Progressive refinement for high-quality results with interactive performance
    • Fast volume rendering with gradient shading and support for implicit isosurfaces
  • Known Issues
    • Minor memory leak during rendering
    • Memory leak when modifying volumes and geometries (changes to transfer functions are managed)
    • View does not always refresh with correct bounds and aspect ratio. workaround: click the reset camera button in ParaView
    • Only uniform grids can be volume rendered
    • Volumes and geometry can not yet be combined. Currently if there is a visible volume, geometry will not be rendered.
    • LightKit does not work.
    • Outline representations do not work.

Alpha Release (2015-04-08)

  • New Features
    • Real-time software rendering of polygonal and volumetric datasets
    • Logarithmic performance scaling with data size, supports volumes and geometries up to the memory limitations of a given system while maintaining interactive performance
    • Support for existing ParaView filters and materials
    • Ambient Occlusion shading
