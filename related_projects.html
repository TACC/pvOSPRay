Projects using / related to pvOSPRay

This page gives a brief (and incomplete) list of other projects that make use of pvOSPRay, as well as a set of related links to other projects and related information.

If you have a project that makes use of pvOSPRay and would like this to be listed here, please let us know.

Projects that use pvOSPRay

"ParaView OSPRay Plugin"

The ParaView OSPRay Plugin (http://tacc.github.io/pvOSPRay/) is based on the vtkOSPRay library.

Projects that pvOSPRay makes use of

  • The vtkOSPRay VTK Interface (tacc.github.io/vtkOSPRay)
  • The OSPRay Ray Tracer (ospray.github.io)
  • The Embree Ray Tracing Kernel Framework (embree.github.io)
  • The Intel® SPMD Program compiler (ISPC, ispc.github.io)
  • OSPRay's "sister project" is the OpenSWR software rasterizer, a new high-performance, highly-scalable, software OpenGL implementation that runs on host CPUs.
